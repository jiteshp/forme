Forme is a responsive business theme. Features include: Jetpack featured posts, custom Google Fonts, sidebar widget area & landing page template.

== Installation ==

Download the theme zip file from https://github.com/jiteshp/forme, unzip it and move the unzipped contents to the wp-content/themes folder of the WordPress installation directory.
Activate and start customizing the theme.


== Description ==
This is a free, responsive, translation-ready business theme.

The important features include -
Jetpack Featured Posts
Custom Google Fonts
Sidebar Widget Area
Landing Page Template
5 Types Of Page Templates
